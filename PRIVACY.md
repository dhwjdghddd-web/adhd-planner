# 개인정보 처리방침 (Privacy Policy)

_최종 수정: 2026-06-30_

이 앱(ADHD 플래너)이 어떤 데이터를 다루는지 설명합니다. 앱은 광고를 표시하지
않으며, 데이터를 판매하거나 광고 목적으로 제3자에게 제공하지 않습니다.

## 수집·저장하는 데이터

계정과 동기화를 위해 다음을 사용합니다(모두 Google Firebase).

- **계정 식별자**: 앱 첫 실행 시 **익명 계정**이 자동 생성됩니다(이름·이메일
  없이 임의의 식별자만). 선택적으로 **Google 계정을 연결**하면 기기를 바꿔도
  데이터가 유지되며, 이때 Google 계정의 이메일/표시 이름을 인증에 사용합니다.
- **사용자가 입력한 콘텐츠**: 하루 구간(블록), 체크리스트 항목, 메모, 하루
  체크인(기분·에너지 1~5 자가 기록), 앱 설정. 이 데이터는 본인 계정 영역
  (`users/{본인 uid}`)에만 저장되며, 보안 규칙상 **본인만 읽고 쓸 수 있습니다.**
- **진단/오류 정보**: 앱이 비정상 동작할 때 Firebase Crashlytics가 오류·기기
  종류·앱 버전 등 진단 정보를 수집합니다(출시 빌드에 한함). 디버그 빌드에서는
  수집하지 않습니다.

## 권한

- **마이크(RECORD_AUDIO)**: **음성 메모** 기능에서 말한 내용을 글자로 받아쓰기
  위해서만 사용합니다. 음성 인식은 기기 운영체제의 음성 인식 기능을 통해
  이루어지며, 앱은 **음성 오디오 자체를 저장하거나 전송하지 않습니다**(받아쓴
  텍스트만 메모로 저장). 운영체제의 음성 인식 처리 방식은 기기 설정을 따릅니다.
- **알림·정확한 알람(POST_NOTIFICATIONS, SCHEDULE_EXACT_ALARM 등)**: 구간 시작
  알람과 리마인더를 정시에 울리기 위해 사용합니다.

## 데이터 보관·삭제

- 데이터는 계정에 연결되어 클라우드(Firebase)와 기기 캐시에 보관됩니다.
- 앱 내 **설정 → "계정·데이터 삭제"**에서 모든 데이터와 계정을 영구 삭제할 수
  있습니다(되돌릴 수 없음).

## 처리 위탁

- 데이터 저장·인증·진단은 Google Firebase(Cloud Firestore, Firebase
  Authentication, Crashlytics)를 통해 처리됩니다. Google의 데이터 처리에 대해서는
  Google 개인정보처리방침을 참고하세요: https://policies.google.com/privacy
